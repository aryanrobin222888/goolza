# goolza ⚽

> The ultimate platform for live football scores, match schedules, and real-time updates.

goolza is a modern, high-performance web application designed to provide football fans with an immersive experience. Built with Next.js and Tailwind CSS, it features a sleek, futuristic design with a focus on usability and speed.

## ✨ Features

- **Live Match Schedule:** Real-time updates for matches happening today, yesterday, and tomorrow.
- **Futuristic Design:** "Brilliant" card aesthetics with an Emerald/Black theme and glassmorphism effects.
- **Mobile-First Experience:** Fully responsive layout optimized for all devices.
- **Arabic Localization:** Built from the ground up with RTL support and the 'Cairo' font family.
- **SEO Optimized:** Rich metadata, Open Graph support, and structured data for search engines.
- **Comprehensive Coverage:** Detailed views for major leagues like the Premier League, La Liga, and more.

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS + Framer Motion (Animations)
- **Icons:** Lucide React
- **Data Fetching:** Axios / SWR (Custom Hooks)
- **Font:** Google Fonts (Cairo, IBM Plex Sans Arabic)

## 🚀 Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/goolza.git
    cd goolza
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
